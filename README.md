# Practicando-React
En mi FEM, me han propuesto crear una API para gestión de clientes; usando CLAUDE CODE he generado el Frontend, con React, Vite y Tailwind.
La carpeta workspace contiene la lógica Java del servidor y la gestión de clientes, es el BACKEND.
El resto de archivos son la estructura del FRONTEND, archivos de React y JavaScript que manejan la conexión y el control de los datos.

Para información más detallada sobre el proyecto consultar el archivo CLAUDE.md
